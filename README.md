PhonePe Transaction Data Analysis Dashboard

An interactive **Power BI Dashboard** built to analyze PhonePe transaction data and provide meaningful business insights through dynamic visualizations, KPI tracking, and trend analysis.

---

## Project Overview

The **PhonePe Transaction Insights Dashboard** helps visualize key transaction metrics, monitor payment performance, analyze customer demographics, and identify usage patterns. The dashboard enables stakeholders to make data-driven decisions using interactive reports.

## Objectives

- Analyze overall transaction performance.
- Monitor Month-over-Month (MoM) growth.
- Track payment success rates.
- Identify top users based on transaction value.
- Understand customer demographics.
- Compare Weekday vs Weekend transaction trends.
- Build an interactive dashboard for business users.

---

## Dashboard Features

### KPI Cards

- Total Transactions
- Total Transaction Value
- Unique Users
- Successful Transaction Rate
- MoM Transaction Growth
- MoM Transaction Value Growth

### Interactive Filters

- Month Filter
- Payment Status Filter

### Visualizations

- Transaction Trend Over Time
- Service-wise Transaction Value Analysis
- Age Segment Contribution
- Weekday vs Weekend Usage
- Top 5 Users by Transaction Value
- Payment Status Analysis

## Business Insights

- Weekdays recorded the highest number of transactions.
- Loan services generated the highest transaction value.
- Millennials contributed the largest share of users.
- Transaction value trends can be monitored monthly.
- Payment status analysis helps identify failed and pending transactions.
